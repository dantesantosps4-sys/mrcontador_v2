# 💰 MR Contador v2

Sistema financeiro com IA que ajuda usuários a controlar gastos, reduzir despesas e tomar decisões inteligentes.

---

## 🚀 Funcionalidades

- 🔐 Sistema de login e cadastro
- 📊 Dashboard financeiro
- 📅 Filtro mensal
- 📈 Gráfico por categoria
- 🤖 IA com 60 perguntas financeiras
- ⚠️ Alertas inteligentes de gastos
- 🧠 Sugestões automáticas
